# About
A one-button android morse keyboard.  

# Usage
Basic usage as with any other keyboard, use a search engine if you're struggling  
  
Swipe gestures:
- Down-left: Backspace dotdash
- Left: Backspace letter
- Up-left: Backspace word
- Down-right: Enter
- Right: Space
- Up-right: CapsLock
- Up: Caps
- Down: Newline
